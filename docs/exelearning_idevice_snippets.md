# Diccionario de Inyección: Snippets XML AUTÉNTICOS (eXeLearning v4)

Este documento contiene la estructura XML **exacta, original y completa** (`<odeComponent>`) de los 14 iDevices presentes en eXeLearning v4.0.0, extraída de una base funcional.

A diferencia de los resúmenes teóricos, estos bloques contienen TODO el andamiaje HTML interno y parámetros JSON (como diccionarios y traducciones `msgs`) que el motor JavaScript espera encontrar para no fallar durante la renderización o exportación del sitio web.

> [!CAUTION]
> Cuando construyas tu generador (en Node, Python, etc.), debes inyectar estos bloques **literalmente como están aquí**. Reemplaza **nicamente** los textos y las cadenas `UUID-PAGINA`, `UUID-BLOQUE`, `UUID-IDEVICE` y `UUID-EVALUACION` (en los juegos evaluativos). No recortes elementos del `htmlView` ni borres variables del `jsonProperties` o el componente web fallará en tiempo de ejecución.
